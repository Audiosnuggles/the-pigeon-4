The Pigeon (v0.5)
The Pigeon is an experimental, web-based synthesizer and sequencer that merges visual drawing with additive sound synthesis. Draw melodies, modulate textures via an interactive Trace Pad, and synchronize your jam setup directly with Ableton Live or external hardware.

‚ú® Features
Visual Sequencing: 4 tracks for freeform melody drawing. The Y-axis determines pitch, while the X-axis represents the timeline.

Additive Engine: Choose between Sine, Square, and Sawtooth oscillators per track.

Interactive Trace Pad ("Poke Me"): A live performance pad for instant sound triggering and X/Y modulation of effects.

6-Unit FX Rack:

Delay: Classic echo with feedback control.

Reverb: Convolution reverb (Algorithmic Impulse).

Vibrato: LFO-based pitch modulation.

Filter: Warm Moog-style 24dB Ladder Filter with soft saturation.

Stutter: Rhythmic gate for chopped-up sounds.

Fractal: A modifier for Chaos (stroke scattering) and Morph (oscillator distortion).

Studio Sync: Professional MIDI Clock support (EXT SYNC) including phase management (Pause/Continue) for DAWs like Ableton Live.

Scale Quantization: "Harmonize" function to snap drawn notes perfectly to musical scales (Major, Minor, Pentatonic).

Data Export: Save your entire session as a JSON set or render your loop as a high-quality WAV file.

üöÄ Installation & Launch
Since The Pigeon is built on modern web technologies (Web Audio API, ES Modules), no installation is required, but a local web server is needed for file imports.

Clone the repository:

Bash
git clone https://github.com/your-username/the-pigeon.git
Start a local server (e.g., using VS Code "Live Server" or Python):

Bash
python -m http.server 8000
Open http://localhost:8000 in a modern browser (Chrome or Edge recommended for Web MIDI).

üéπ MIDI Sync Setup (Ableton Live)
Virtual Cable: Enable the IAC Driver (macOS) or install loopMIDI (Windows).

Ableton: In MIDI Preferences, activate the Sync output for the virtual cable.

The Pigeon: Click EXT SYNC, select the cable from the dropdown menu, and hit Play in Ableton.
